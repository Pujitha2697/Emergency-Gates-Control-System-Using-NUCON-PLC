# Emergency-Gates-Control-System-Using-NUCON-PLC
Emergency Gates Control System developed using NUCON PLC and CODESYS Ladder Logic during my ECIL internship. The project focuses on industrial automation, PLC programming, Digital &amp; Analog I/O modules, FPGA testing, and emergency gate control.

# Project Overview
This project was completed during my internship at Electronics Corporation of India Limited (ECIL), Hyderabad.
The project demonstrates an industrial automation system using NUCON PLC and CODESYS to automate emergency gate operations. The PLC monitors various inputs such as emergency switches, sensors, and push buttons to safely control gate movement during emergency situations.

# Objectives
- Develop an automated emergency gate control system.
- Learn PLC programming using CODESYS.
- Test Digital and Analog I/O modules.
- Perform FPGA board testing.
- Improve industrial safety using automation.
